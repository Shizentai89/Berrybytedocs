BerryByte offers game server hosting and virtual servers backed by [terabit-scale DDoS protection](https://berrybyte.net/ddos), latest-gen Ryzen 9 hardware, a white-glove support team. All bundled with a commitment to continously build and push new features, updates, and changes.

This site contains useful guides & tips on how to run various types of game
servers (including Minecraft, Rust and more to come), as well as running VPSs.

These guides are designed for BerryByte users, however the same guides can be applied most certainly be applied in other environments.

<Warning>
  This documentation is a work-in-progress. There may be errors or typos in the
  guides, however we're constantly working to improve them. Suggest edits at the bottom of the guide!
</Warning>
